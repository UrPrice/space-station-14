# Weapon Analyzer

Утилита командной строки для анализа характеристик оружия и расчета урона по различным целям. Инструмент симулирует попадания с учетом сопротивлений, модификаторов урона,характеристик брони конкретных мобов, но без учёта разброса и нетривиальных эффектов, после чего выгружает статистику в CSV-формат.

## Основные возможности

* **Извлечение данных об оружии:** Автоматический парсинг `GunComponent`, батарей, магазинов и типов патронов (хитскан, проджектайлы).
* **Симуляция урона:** Расчет реального урона (Actual Damage) по конкретным целям (например, `MobHuman`) с учетом встроенных резистов и модификаторов (`DamageModifierSet`).
* **Расчет TTK и BTK:** Автоматическое вычисление Time-to-Kill (времени на убийство) и Shots-to-Kill (количества выстрелов до наступления стадии *Critical*).
* **Анализ Stamina:** Учет урона по выносливости (Stamina Damage) и расчет Stamina DPS.
* **Пакетная выгрузка:** Генерация CSV-файлов под каждую указанную цель.

## Использование

Инструмент запускается через интерфейс командной строки. По умолчанию он анализирует урон по цели `MobHuman` и сохраняет результат в `weapons.csv`.

```bash
# Запуск со стандартными значениями аргументом
dotnet run --project Content.WeaponAnalyzer

# Запуск с указанием конкретных целей и пути сохранения
dotnet run --project Content.WeaponAnalyzer -- -t MobHuman MobHumanSyndicateAgentNukeops

```

### Аргументы командной строки

| Флаг | Полный параметр | По умолчанию | Описание |
| --- | --- | --- | --- |
| `-o` | `--out` | `Content.WeaponAnalyzer/weapons.csv` | Путь к выходному `.csv` файлу. Инструмент автоматически добавит имя цели в название файла (например, `weapons_MobHuman.csv`). |
| `-t` | `--targets` | `MobHuman` | Список ID прототипов целей для анализа (мобов). Урон будет рассчитываться индивидуально для каждой цели. |
|  | `--keep-errors` | `false` | Если флаг установлен, в итоговый список попадут виды оружия, при парсинге которых произошли ошибки (например, не найдена батарея или патроны). |

## Структура выходных данных (CSV)

Сгенерированный файл содержит следующие колонки:

* **PrototypeId:** ID прототипа оружия.
* **FireRate:** Скорострельность.
* **GunMagazine:** ID используемого магазина (если применимо).
* **Capacity:** Вместимость (патронов или зарядов батареи).
* **GunChamber / CartridgeAmmo:** Данные о патроннике и типе боеприпаса.
* **IgnoreResistances:** Флаг `true`/`false`, игнорирует ли оружие сопротивление цели.
* **ActualDamage:** Фактический урон за выстрел (после применения модификаторов цели).
* **TDPS:** Фактический урон в секунду (True DPS).
* **SummaryDamage:** Суммарный урон со всего магазина/заряда.
* **StaminaDamage / StaminaDPS / SummaryStaminaDamage:** Показатели урона по выносливости, в виде аналогичном трём предыдущим по обычному урону.
* **Health:** Количество здоровья цели до перехода в крит (DamageToKill).
* **TTK:** Time-to-Kill в секундах.
* **BTK:** Bullets-to-Kill (количество выстрелов/ударов для убийства).

## How to use

Чтобы открыть итоговый файл `weapons.csv` при помощи `Excel` нужно открыть программу `Excel` перейти на вкладку Данные > Получение данных > Из файла > Из текстового/CSV-файла > Выберите файл.

Чтобы открыть итоговый файл `weapons.csv` при помощи `Libre Office` - откройте файл.
